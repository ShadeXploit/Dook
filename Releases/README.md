# Compiled Releases

## Purpose

This folder contains **pre-compiled, ready-to-go executable versions** of Dook for different operating systems. 

If you don't want to deal with Python installation, dependencies, or compilation - this is for you!

### Simply:
1. **Download** the version for your operating system (Windows, macOS, or Linux)
2. **Run** the executable
3. **Done!** No installation required

---

## Available Operating Systems

- **Windows** - Direct `.exe` executable
- **macOS** - Application bundle ready to run
- **Linux** - Ready-to-use binary

No Python needed. No dependencies to install. Just download your OS version and go!

---

*Each release is fully compiled and tested for your convenience.*
